Data strunctues List includes a) Stack, b) Queue, c) Deque (Double ended queue), d) Linked list, e)HashMap/ HashSet, f) BinaryTree, g) Heap (Priority Queue), h) Graph, i) Trie (Prefix Tree), j) Dynamic Programming (2D Table)

1. Stack -- Use arrays with push() and pop().

2. Queue -- Use arrays with push() (enqueue) and shift() (dequeue).

3. Deque (Double-Ended Queue) -- Use arrays with unshift() and shift() for front operations and push() and pop() for rear operations.

4. Linked List -- Use objects or custom classes to represent nodes and pointers.

5. HashMap/HashSet -- Use JavaScript objects or Map/Set.

6. Binary Tree -- Represent nodes as objects or classes with left and right pointers.

7. Heap (Priority Queue) -- Use arrays and write helper functions for heapify operations (or use a library like heapify-js).

8. Graph -- Use adjacency lists or matrices for representation.

9. Trie (Prefix Tree) -- Use nested objects for nodes.

10. Dynamic Programming (2D Table) -- Use arrays for memoization and state storage.
